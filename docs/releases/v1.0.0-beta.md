# Nebula-Stream v1.0.0-beta

Release name: **The First Constellation**

## Highlights

- Event-driven orchestration foundation built on Go and NATS patterns.
- Protobuf/gRPC control-plane schema baseline.
- Workflow parsing and execution skeleton (sequential, parallel, branching).
- WASM runtime integration points (Wasmtime-oriented) with sandbox hooks.
- AI worker placeholders and observability groundwork.
- Dashboard and React Flow integration starting point.

## Operational Additions

- Basic Docker Compose setup for local dependency bootstrapping.
- GitHub Actions CI workflow for backend module tests.
- Initial benchmark and security audit documentation.

## Known Gaps

- Production-grade runtime isolation hardening is still in progress.
- Advanced retry policies, durable state backends, and full API coverage are pending.
- UI currently represents foundational scaffolding.
