# Discussions Playbook

The following discussion threads are pinned starter channels for growth:

- Showcase: https://github.com/erciktiburak/Nebula-Stream/discussions/1
- Roadmap voting: https://github.com/erciktiburak/Nebula-Stream/discussions/2

## Weekly routine

1. Re-share the showcase thread and highlight one community project.
2. Summarize roadmap poll feedback and convert top items into issues.
3. Post one short dev update with screenshot/GIF to keep activity visible.
